Detailed Description

Defines two dimensional double region.

Regions are x1 <= x < x2

Infinite regions are flagged by setting

  • x1 = kOfxFlagInfiniteMin
  • y1 = kOfxFlagInfiniteMin
  • x2 = kOfxFlagInfiniteMax
  • y2 = kOfxFlagInfiniteMax
