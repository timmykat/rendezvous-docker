kOfxParamPropGroupOpen
Prev Chapter 17. Properties By Name Next

Name

kOfxParamPropGroupOpen — Whether the initial state of a group is open or closed in a hierarchical layout.

Synopsis

#include "/ofxParam.h"
#define kOfxParamPropGroupOpen "OfxParamPropGroupOpen"

Type

kOfxParamPropGroupOpen is of type int, and has only a single dimension.

Property Set

This is a property that belongs to the plugin parameter descriptor (read/write) and instance (read only).

Default

The default value is 1.

Valid Values

0 or 1

Description

Whether the initial state of a group is open or closed in a hierarchical layout.

This is a property on parameters of type kOfxParamTypeGroup, and tells the group whether it should be open or closed by default.
